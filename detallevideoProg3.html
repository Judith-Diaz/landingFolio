Descripcion del proyecto

Este sistema web de gestión para una clínica fue desarrollado en C# con base de datos SQL Server. Permite administrar de forma integral a los médicos, pacientes y turnos, brindando herramientas completas para el alta, baja, modificación y listado (ABML) de cada uno. Cuenta con un sistema de usuarios con distintos roles, como Administrador y Médico, cada uno con funcionalidades específicas. Los médicos pueden visualizar exclusivamente sus turnos asignados, agregar observaciones médicas y registrar la asistencia de los pacientes. El sistema está diseñado con una interfaz clara y funcional, orientada a facilitar el uso tanto para el personal médico como administrativo.

Tecnologías Utilizadas

Backend: C#, .NET, ASP.NET.

Frontend: HTML, CSS.

Base de Datos: SQL Server.
