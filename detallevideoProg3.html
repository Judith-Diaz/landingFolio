Descripcion del proyecto

✔️ Administración de médicos, pacientes y gestión de turnos. ✔️ Gestión de usuarios con roles: Administrador y Médico. ✔️ ABML (Alta, Baja, Modificación, Listado) para pacientes, médicos y turnos. ✔️ Visualización de turnos específicos para cada médico, con opción de agregar observaciones y marcar asistencia del paciente.

Tecnologías Utilizadas

✔️ Backend: C#, .NET, ASP.NET. ✔️ Frontend: HTML, CSS. ✔️ Base de Datos: SQL Server.
