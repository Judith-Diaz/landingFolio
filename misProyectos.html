Aplicación de gestión de películas

Desarrollamos una aplicación de gestión de películas en Python utilizando Tkinter para la interfaz gráfica y SQLite como base de datos. Esta aplicación permite realizar operaciones de ABML (Alta, Baja, Modificación y Listar) de películas, manejando información como nombre, duración, género, clasificación y calificación. La tabla principal muestra los registros de las películas cargadas , input, y botones que nos permiten realizar ABML
