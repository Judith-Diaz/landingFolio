Descripcion del proyecto

Bienvenido a la aplicación de gestión de películas,con ella podras añadir, modificar, eliminar y listar películas. Desarrolle una aplicación de gestión de películas en Python utilizando Tkinter para la interfaz gráfica y SQLite como base de datos. Esta aplicación permite realizar operaciones de ABML (Alta, Baja, Modificación y Listar) de películas, manejando información como nombre, duración, género, clasificación y calificación. La tabla principal muestra los registros de las películas cargadas , input, y botones que nos permiten realizar ABML

Tecnologías Utilizadas

Backend: Python.

Visual: Tkinter.

Base de Datos: Sqlite3.
